Track Events

When you need to track events on items in your collection, like clicks or hover states, a Collection View or Composite View is the class of choice.

When To Use

Use a Collection View when you need to render the list within a single element, like a ul. If you have a more complex DOM structure around your list, like a wrapping table, then use a Composite View.

The reason for this distinction is that Composite Views can have a template associated with them. Within the template you can specify where to render the list. Collection Views only have an el, and the default render function will simply render the list within that element.
